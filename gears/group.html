group¶

Groups data received at the input into Queue -s of certain size.

group(din, size: Uint) → Queue[din]¶

Number of elements in the output Queue is specified by the data received at size input interface.

size = drv(t=Uint[3], seq=[3, 4])

drv(t=Uint[4], seq=[1, 2, 3, 4, 5, 6, 7]) \
    | group(size=size) \
    | check(ref=[[1, 2, 3], [4, 5, 6, 7]])
